Topic : XML ELEMENTS

  • XML Stand for eXtesible Markup Language
  • XML is a markup language much like HTML
  • XML was designed to store and transport data
  • XML was designed to be self-descriptive

Difference Between XML and HTML

XML HTML
XML is designed to carry data emphasizing on what type of data it is. HTML is designed to display data emphasizing on how data looks
XML tags are not predefined HTML tags are predefined
HTML is about displaying data,hence it is static XML is about carrying information,which makes it dynamic.

	</child>

XML ELEMENTS

  • The XML elements are the basic building block of the XML document.
  • It is used as a container to store text elements, attributes, media objects etc.
  • Every XML documents contain at least one element whose scopes are delimited by start and end tags or
    in case of empty elements it is delimited by an empty tag.

Syntax 

   <element-name attributes> Contents...</element-name>

Rule to define XML elements

  • An element can contain alphanumeric values or characters. But only three special characters are required in the names these are hyphen, underscore and period.
  • Names are case sensitive. For example address, Address, aDDress are different names.
  • Both start and end tags for elements need to be same.

Example 


	<?xml version = “1.0”?>
	<contactinfo>
	<address category = “college”>
	<name> Ayan </name>
	<College> ICLES MJ </College>
	<mobile>2345456767</mobile>
	</address>
	</contactinfo>

Output

Ayan
ICLES MJ
